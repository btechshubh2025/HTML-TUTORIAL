What are HTML Attributes?

Attributes provide additional information about HTML elements.

Common HTML Attributes:

  • href: Specifies the URL of a link.

    Visit Google

  • src: Defines the image source.
  • width & height: Set the dimensions of an image.
  • alt: Provides alternative text for an image.

  • style: Adds inline CSS styles.

    Styled text example

  • lang: Specifies the language of the document.

    This is an English paragraph.

  • title: Displays additional information on hover.
