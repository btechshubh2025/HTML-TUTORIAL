ASCII Character Set

ASCII includes: A-Z, a-z, 0-9, and special characters like ! $ + - ( ) @ < >

ANSI (Windows-1252) Character Set

ANSI supports ASCII characters and extends with special symbols like € ™ ©

ISO-8859-1 Character Set

ISO-8859-1 extends ASCII but excludes values from 128-159, supporting characters like é ô ü

UTF-8 Character Set

UTF-8 supports ASCII, ISO-8859-1, and extends beyond, allowing characters like ह こ 你
